# Steps to create a NodeJS app

> Setting up the project directory
 1. mkdir project_name
 2. cd project_name

 > Initializing your project
 3. npm init
	 Enter the project details that follows.
 * For Window Users:
	 4. echo.>index.js
 * For Linux Users:
	 4. touch index.js
5. Write server code in index.js file as given.

> Start server
6. node index.js
